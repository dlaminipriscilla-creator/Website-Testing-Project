# Website Testing Project

## Project Overview
This project involved manual testing of a live business website to identify functional and UI/UX defects before production release.

## Scope of Testing
- Functional Testing
- UI/UX Testing
- Exploratory Testing
- End-to-End Testing

## Deliverables
### Test Cases
Contains structured manual test cases written in Excel.

### QA Report
Contains:
- Defect descriptions
- Screenshots
- Steps to reproduce
- Expected vs Actual Results

## Tools Used
- Microsoft Excel
- Google Docs
- Google Sheets

## Skills Demonstrated
- Bug Reporting
- Test Documentation
- Attention to Detail
- QA Analysis
